# DNA to RNA Transcription

## Overview

This project converts a DNA sequence into its corresponding mRNA sequence by replacing thymine (T) with uracil (U).

## Example

Input:
ATGCCGTTA

Output:
AUGCCGUUA

## Biological Background

During transcription, RNA polymerase synthesizes mRNA from a DNA template strand. In RNA, uracil (U) replaces thymine (T).

## Python Concepts Used

- Functions
- String manipulation
- User input
- bioinformatics sequence analysis
