# Bioinformatics Python Projects

A collection of beginner-friendly bioinformatics projects developed using Python. This repository focuses on DNA, RNA, and sequence analysis applications commonly used in biotechnology and bioinformatics.

## Projects

### DNA to RNA Transcription
Converts a DNA sequence into its corresponding mRNA sequence.

Example:

DNA: ATGCCGTTA

RNA: AUGCCGUUA

### GC Content Calculator
Calculates the percentage of G and C nucleotides in a DNA sequence.

### Reverse Complement Generator
Generates the reverse complement of a DNA sequence.

### Nucleotide Counter
Counts the occurrence of A, T, G, and C in a DNA sequence.

### RNA to Protein Translation
Translates mRNA codons into amino acid sequences.

### FASTA File Analyzer
Reads FASTA files and provides:
- Sequence length
- GC content
- Nucleotide composition

## Technologies Used

- Python 3
- Bioinformatics Algorithms
- File Handling
- Sequence Analysis
